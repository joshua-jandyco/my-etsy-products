Example Usage of MyEtsyProducts
  1. Include the my-etsy-products.js script:
    <script src="my-etsy-products.js"></script>
  2. Include the base style css
    <link rel="stylesheet" href="etsy-style-base.css" />
  3. Include the pagination style css
    <link rel="stylesheet" href="etsy-style-pagination.css" />
  4. Choose either the cell or row style css
    <link rel="stylesheet" href="etsy-style-cell.css" />
  5. Optionally include the css-spiners spinner css
    <link rel="stylesheet" href="http://css-spinners.com/css/spinner/spinner.css" type="text/css" />
  6. Call myEtsProducts.init, passing the following:
    1. The id of the container the products will be in. (E.G. 'my-products')
    2. The id of your Etsy store
    3. The name of your Etsy store as it appears in the URL
    4. The number of items per page to display
    5. The page to start on
    6. Your Etsy API key. Get one here
    <script>
    window.onload = function() {
        var options = {itemsPerPage: 10, descriptionLength: 350}
        myEtsyProducts.init("my-products","yourstorId", "yourStoreNameAsSeenInUrl" 1, 'your-api-key-here', options);
    }
</script>
